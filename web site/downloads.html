Downloads

To see the differences between the release versions, see the release notes. If you intend to do any benchmarking, please read the Benchmarking Guidelines. The downloads are for Win32 only. Available releases:

Binaries (Win32)

  • Current Release: Strongtalk-2.0.zip
  • Strongtalk-1.1.2.zip (2.9MB) Release Date: August 9, 2006

    If you want to apply all the differences from the 1.1.1 release yourself, they are here: Strongtalk-1.1.2-diffs.zip (11K)

  • Strongtalk-1.1.1.zip (2.9MB) Release Date: July 4, 2002
    If you have 1.1 and just want the code patches (without doc changes), download this simple file-in: Strongtalk-1.1-patch.dlt.

    If you want to apply the all differences from the 1.1 release yourself, they are here: Strongtalk-1.1.1-diffs.zip (11K)

  • Strongtalk-1.1.zip (2.9MB) Release Date: July 1, 2002

    If you want to apply the differences from the 1.0 release yourself (not recommended), they are here: Strongtalk-1.1-diffs.zip (420K)

  • Strongtalk-1.0.zip (2.5MB) Release Date: July 1, 2002

    This is the original release from Sun Microsystems. There is rarely a reason to download this, since it does not work on Windows XP and has plenty of other niggling problems.

The software is free, and open source, under a BSD-style license. Sun's license is included in the download, however, if you'd like to look at it first, it is here:

Source Code

The source code repository (under Subversion) can be found at: http://code.google.com/p/strongtalk/source.

Pre-packaged Source

The source code for the Strongtalk VM has just been released as of September 10, 2006. Development was halted on Strongtalk in the late 90s, so at the moment the source only builds and runs correctly using older versions of MS C++, and also requires Borland Turbo Assembler for the small remaining amount of assembly code (parts of the interpreter).

  • strongtalk-source-2.0.zip (4.9MB)
  • strongtalk-source-1.0.1.zip (3.7MB) - This is the original source release from Sun Microsystems. The version number for this does not match any binary release, since this version does not build.
